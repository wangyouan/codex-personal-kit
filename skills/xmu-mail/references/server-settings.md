# XMU Mail Server Settings

Official source: https://inc.xmu.edu.cn/inc2024/info/1011/1042.htm

Published by Xiamen University Information and Network Center on 2022-07-06.

## Endpoints

- Webmail: https://mail.xmu.edu.cn
- IMAP server: imap.xmu.edu.cn
- POP3 server: pop3.xmu.edu.cn
- SMTP server: smtp.xmu.edu.cn

## Ports

- IMAP SSL: 993
- IMAP non-SSL: 143
- POP3 SSL: 995
- POP3 non-SSL: 110
- SMTP SSL: 465
- SMTP non-SSL: 25

Use SSL by default.

## Limits and Notes

- Attachment size limit: 50 MB.
- Single-message size limit: 70 MB.
- Do not use `xmu.edu.cn` as the mail server host.
- Check mailbox quota periodically.
- Spam is retained for 30 days.
- Accounts unused for one year may be deleted.
- Bulk sending may lock the mailbox; contact the university service desk for legitimate mass-mail needs.
